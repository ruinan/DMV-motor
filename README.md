## 当前决策原则

- 先把最大 scope 定清楚，再讨论实现
- 先拆功能，再决定具体技术方案
- 所有设计都必须服务“提高 M1 笔试通过率”
- 不把产品扩展成安全教育平台或大而全学习系统
